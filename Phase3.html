Phase 3 - Vm252 Debugger

Changes from phase 2 to phase 3:

  • Added the feature for the user to upload a file from the computer and for the file to display line-by-line in the text box on the top right.
  • Added functionality to run the debugger on an uploaded file.
  • Added functionality to the “Pause” and “Resume” buttons.
  • Added functionality to go through the debugging process single-step
  • Added functionality to display the contents of memory as a program runs.
